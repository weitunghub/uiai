# Browser Use
是一个开源库，用Python写的，专门让AI来控制浏览器完成各种任务。它结合了自然语言处理、Playwright（一个浏览器自动化工具）和大型语言模型（比如GPT-4、Claude等），让AI能够像人类一样浏览网页、点击链接、填写表单、提取信息，甚至还能做更复杂的操作，比如自动找工作、写文档、保存文件等等。

项目地址：
**https://github.com/browser-use/browser-use**
